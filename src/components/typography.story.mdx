# Design system

Design system is based on Bootstrap 4 and provides reusable elements defined directly by tags or additionally by class names.

<!--- This is your Story template function, shown here in React -->

## Typography

For accomplishing seamlessly same looking across whole application.

### Display

Extending `H*` tags by four huge variants using `display-*` class name.
